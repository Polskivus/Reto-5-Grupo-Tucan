# GastroLab - Reto 5

# Descripción

GastroLab es una aplicación web desarrollada como parte del Reto 5 del ciclo de Desarrollo de Aplicaciones Multiplataforma (DAM).

El objetivo del proyecto es dar solución a los problemas de organización y gestión de información de un centro de formación culinaria. Actualmente, recetas, menús y datos administrativos se encuentran distribuidos en distintos formatos, dificultando su acceso y mantenimiento.

Para resolver esta situación, se ha diseñado una aplicación web conectada a una base de datos que permite centralizar toda la información en un único sistema.

# Objetivos del proyecto

- Centralizar la gestión de recetas, menús y personal.
- Mejorar la organización interna del centro.
- Digitalizar procesos administrativos.
- Facilitar el acceso a información nutricional.
- Aplicar buenas prácticas de desarrollo y seguridad.

# Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- Python
- MySQL
- Pytest
- Git

## Funcionalidades principales

### Gestión de recetas
- Crear, consultar y modificar recetas.
- Gestionar ingredientes.
- Organizar menús.
- Registrar dietas y datos nutricionales.

### Gestión administrativa
- Gestión de usuarios y empleados.
- Gestión de contratos.
- Gestión de nóminas.

### Seguridad
- Sistema de roles y permisos.
- Contraseñas cifradas.
- Protección frente a inyecciones SQL.

## Base de datos

La base de datos incluye tablas relacionadas con:

- Usuarios
- Trabajadores
- Contratos
- Nóminas
- Recetas
- Ingredientes
- Dietas

Además, se han implementado roles, procedimientos almacenados, funciones y triggers para mejorar la seguridad y automatización del sistema.

# Testing

Se han realizado pruebas unitarias con Pytest para comprobar:

- La creación correcta de la aplicación.
- La configuración del entorno de pruebas.
- El funcionamiento básico del sistema.

# Metodología de trabajo

El proyecto se ha desarrollado en equipo utilizando metodología ágil basada en sprints, realizando entregas parciales y revisiones continuas para mejorar el resultado final.

# Conclusión

Este proyecto ha permitido aplicar conocimientos de programación, bases de datos, entornos de desarrollo , sostenibilidad , digitalizacion y trabajo en equipo en un entorno similar al profesional.
